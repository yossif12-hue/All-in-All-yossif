# All-in-All — Static demo (yossif folder)

هذا المجلد يحتوي على نسخة من مشروع العرض باسم `yossif`.

ملفات:
- `index.html` — الصفحة الرئيسية
- `css/styles.css` — أنماط
- `js/app.js` — السكريبت

لتشغيل محلياً:
```bash
cd yossif
python3 -m http.server 8000
# افتح http://localhost:8000
```

ملحوظة: Geolocation يعمل عبر `localhost` أو HTTPS.
